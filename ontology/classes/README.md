# Concept Classes

| Class | Description | Translation Policy |
|-------|-------------|-------------------|
| Feature | Individual product capabilities available to users depending on their plan. | prefer_adapt |
| Plan | Subscription tiers within a product line, defined by price, limits, and included features. | do_not_translate |
| ProductLine | Top-level product families that group related plans and offerings (e.g. Creator, Studio). | do_not_translate |
